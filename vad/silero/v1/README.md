# Silero VAD v6

Voice Activity Detection model from [snakers4/silero-vad](https://github.com/snakers4/silero-vad).

## Model Details

- **Parameters:** ~260K
- **Sample rates:** 8 kHz and 16 kHz
- **Window sizes:** 256 samples (8 kHz) or 512 samples (16 kHz)
- **Language support:** 6000+ languages
- **ONNX opset:** 16

## Files

| File | Size | Description |
|------|------|-------------|
| `silero_vad.onnx` | 2.2 MB | Full precision VAD model |
